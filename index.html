Para utilizar o sistema:

  1. Lembre-se criar a pasta em /var/lib/asterisk/sounds/ura-cliente;

  2. Realizar o Upload dos audios e converter com o comando:
    sox arquivo.wav -r 8000 -c 1 /var/lib/asterisk/sounds/ura-cliente/arquivo_ura.wav (verificar o nome na linha do read);

  3. Declarar o arquivo no extension.conf:
    #tryinclude custom/ura_cliente.conf;

  4. asterisk -rx 'dialplan reload';

  5. Criar o contexto das filas ou em um arquivo chamandos as macros ou na interface web
